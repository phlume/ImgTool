Image Tool

Create "a banner link" snippet on the fly.

Simply drag this to your tool bar of chrome or firefox, and then click it when you are on a page that you want to generate a link from. The cursor will turn into a "Cross hair" on images you can grab code from.

Once you click the image a new window will appear in the top right corner of your screen. You can then select and copy the image path, the link or the all inclusive html tag to create a new linked banner.
